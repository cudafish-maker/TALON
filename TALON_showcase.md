# T.A.L.O.N. — Tactical Awareness & Linked Operations Network

A self-hosted, encrypted command-and-control platform for teams that need reliable coordination when conventional infrastructure fails. Runs over LoRa radio, I2P, and Yggdrasil — no internet required.

**Features:** tactical map · situation reports · mission management · encrypted chat · document repository · full server admin

**Stack:** Python · Kivy/KivyMD · SQLCipher · PyNaCl · Reticulum
