Minimal SITREP Wireframe

From-scratch SITREP page focused on create, read, append, severity awareness, and server-only deletion. This wireframe intentionally removes incident handoff, document linking, templates, acknowledge, and close controls from the main SITREP surface. Assignment is a single focused popup that only lists currently available operators.

Approval Target

  • Operators create SITREPs and append more information.
  • Operators can assign an available, unassigned operator.
  • Levels and notification counts stay visible.
  • Deletion remains separated and server-only.
  • Create dialog keeps context collapsed and secondary.
