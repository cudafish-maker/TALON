SITREP Template Control Wireframe

This mockup replaces the crowded template button grid with a compact template selector and Apply action. It keeps the composer column narrow enough for smaller physical screens while preventing clipped template labels and the clipped Pick on Map button.

Approval Target

  • Template selection becomes a combo box plus Apply button.
  • The selected template gets a short preview below the selector.
  • Lat/Lon keeps two coordinate fields plus a short Map button.
  • No app code changes are included in this wireframe.

Why This Fix

  • The template control is an option set, so a combo box is a better fit than many command buttons.
  • The composer keeps the narrower minimum widths that prevent the overall window from exceeding physical screens.
  • The selected template remains visible, and the Apply action is explicit.
  • The Map button is short enough to fit beside Latitude and Longitude without truncating.
  • The preview gives confidence about the template without requiring a large button grid.
