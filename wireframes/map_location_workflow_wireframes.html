Map Location Workflow Wireframes

Approval artifact for the location-picker fixes: draft mission geometry stays visible across picker steps, assignments get a map-picked coordinate, SITREP manual lat/lon gains the same map action, and mission selection on the Map page focuses the mission area.

Approval Scope

  • Reuse the existing PySide6 map picker and live map controls.
  • Persist assignment coordinates as the existing core assignment lat/lon fields.
  • Add a SITREP map button beside manual lat/lon fields.
  • Keep the main window within the available screen by allowing pages to shrink.
