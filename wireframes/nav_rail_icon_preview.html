TALON Nav Rail Icon Preview

Preview-only concept for replacing the current Qt standard navigation icons with a deliberate TALON-specific symbol set. This page does not implement any application changes.

Approval Check

  • Every current server nav section has a purpose-built icon.
  • Icons use thin linework to stay legible in the collapsed 22px rail.
  • Active and badge states match the current desktop nav styling.
  • Client mode would use the same common-section icons without server-only entries.
