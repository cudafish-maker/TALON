TALON Map Asset Icon Preview

Preview-only concept for map assets. The symbols borrow the simple frame-and-glyph language operators expect from NATO-style mapping, but this is not a MIL-STD-2525 or APP-6 compliant set.

Approval Check

  • Amber dashed framed icons represent unverified assets.
  • The same symbols scale from dense 24px markers to 48px detail views.
  • No application code changes are included in this preview.
