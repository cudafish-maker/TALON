# Codex Project Instructions

At the start of each new Codex conversation in this repository:

1. Read `CLAUDE.md`.
2. Read `wiki/INDEX.md`.
3. Confirm the current Git branch is `dev` before changing files. If the
   workspace is on another branch, switch to `dev` before edits unless the user
   explicitly asks to work elsewhere.
4. Treat the wiki as the authoritative project state unless it conflicts with higher-priority instructions.
5. When changing features or fixing bugs, follow the wiki update workflows described in `CLAUDE.md`.

These instructions are repo-specific for T.A.L.O.N. and should not be generalized to unrelated projects.
