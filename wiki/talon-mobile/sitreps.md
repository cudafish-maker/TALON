# Mobile SITREPs

Mobile SITREPs are a high-priority field workflow after unlock/enrollment and
map dashboard.

## Required Workflows

- Create SITREP quickly from field dashboard.
- Choose severity level.
- Link asset or mission when available.
- View feed.
- Receive severity overlays.
- Respect opt-in-only FLASH audio.

## Constraints

- Audio playback eligibility comes from core events.
- Startup hydration should not create unread badges for old records.
- Server-only delete remains unavailable on mobile client.
