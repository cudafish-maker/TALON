# Mobile Chat

Mobile chat is planned after core sync and primary field workflows are stable.

## Required Workflows

- View default, mission, custom, and DM channels.
- Send and receive messages.
- Show pending/offline send state.
- Open mission channel from mission detail.

## Constraints

- Server-delete controls are desktop/server-only.
- DMs remain server-readable until core Phase 2b E2E encryption lands.
- Mobile UI must display security state based on core read models.
